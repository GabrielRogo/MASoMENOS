Dos idiomas, un significado
El álgebra tiene su propio idioma. Aprende a traducir frases cotidianas a expresiones algebraicas, y viceversa.
1 · El diccionario del álgebra
Palabras → Operaciones
En lenguaje matemático, muchas palabras cotidianas tienen una traducción directa a operaciones.

Aprender este "diccionario" es la clave para convertir cualquier problema de palabras en una expresión algebraica.
Diccionario álgebra básico
+
sumar · más · aumentar · añadir · ganar · total de
−
restar · menos · disminuir · quitar · perder · diferencia
·
multiplicar · veces · el doble de · el triple de · producto
÷
dividir · la mitad de · repartir · cociente · entre
2 · Ejemplos bidireccionales
La variable como "número desconocido"
Cuando una frase dice "un número", "cierta cantidad" o "algo desconocido", usamos una letra.

La letra más común es x, pero podemos usar cualquiera: n, a, t…

Ejemplo: "tres más un número" → 3 + x
Lenguaje natural ↔ Lenguaje algebraico

3 · ¡Ojo con el orden!
La resta no es conmutativa
En la suma x + 3 = 3 + x (da igual el orden).

En la resta NO:
• "3 menos que x" → x − 3 ✓
• "x menos que 3" → 3 − x ✓

"Tres menos que un número" = el número, quitándole 3 = x − 3
"Un número menos que tres" = 3, quitándole el número = 3 − x
